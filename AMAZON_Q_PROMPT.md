# Amazon Q Prompt: Elegant Pink Birthday Site Design

## Role & Context
You are an expert frontend developer, UI/UX designer and animation specialist. The user wants to transform their Next.js birthday website to match an elegant, mature, romantic aesthetic similar to a luxury birthday celebration design.

## Current State
- Next.js website with cute/playful cat theme
- Gradient backgrounds in pastel colors
- Framer Motion animations

## Desired Design Transformation
Transform the birthday website into a sophisticated, elegant celebration design with a refined pink palette and romantic aesthetic.

## Design Requirements

### Color Palette
- **Primary Pinks**: Blush pink (#FFC0CB), rose pink (#FF7F9B), dusty rose (#D5A6A6)
- **Warm Tones**: Champagne (#F7E7CE), peach (#FFD4C4), soft mauve (#D4A5A5)
- **Soft Backgrounds**: Off-white cream (#FAF6F1), pale pink (#FFE4E1)
- **Accents**: Deep rose (#C85A5A), burgundy (#A85555)
- **Supporting**: Soft gold (#D4AF97), blush lavender (#E8D5E3)

### Visual Style
- Elegant and mature aesthetic (not childish/playful)
- Warm, romantic, and celebratory vibe
- Sophisticated minimalism with warm touches
- Soft lighting effects and subtle glows
- Floating petals and particle animations
- Premium feel with subtle shadows and depth

### Key Design Elements
1. **Hero Section**: 
   - Center-aligned elegant typography with serif fonts
   - Beautiful birthday cake illustration or image (sophisticated, not cute)
   - Soft rose petals or petal animations floating in background
   - Romantic lighting glow effects

2. **Typography**:
   - Serif fonts for headings (e.g., Playfair Display, Lora, or similar)
   - Clean sans-serif for body text (e.g., Inter, Poppins)
   - Elegant color gradients for main heading

3. **Decorative Elements**:
   - Floating rose petals (not emoji)
   - Soft sparkle effects with warm glow
   - Subtle background bokeh or light particles
   - Ornamental dividers or floral borders (minimal)
   - Delicate line elements and decorative dots

4. **Button Styling**:
   - Premium gradient button (rose to burgundy)
   - Smooth hover animations
   - Rounded edges with shadow depth
   - Elegant typography on button

5. **Background & Atmosphere**:
   - Soft gradient backgrounds in warm pinks and creams
   - Subtle texture or bokeh effects
   - Warm ambient lighting simulation
   - Romantic mood through color and composition

### Technical Implementation
- **Framework**: Next.js (already in use)
- **Styling**: Tailwind CSS + custom CSS for gradients
- **Animations**: Framer Motion for smooth transitions
- **Fonts**: Google Fonts (Playfair Display or Lora for headings)
- **Responsiveness**: Fully responsive for all devices
- **Performance**: Optimized animations and images

### Specific Changes
1. Replace playful cat graphics with elegant cake or floral imagery
2. Update color scheme in globals.css and component styles
3. Implement sophisticated typography hierarchy
4. Add floating petal animations instead of emoji floating
5. Create refined button and card components
6. Add ambient bokeh/sparkle background effects
7. Ensure romantic lighting through shadows and glows
8. Update spacing and sizing for premium feel
9. Implement smooth page transitions with elegant animations
10. Add decorative dividers and ornamental elements

### Tone & Voice
- Sophisticated and elegant
- Romantic and warm
- Celebratory but refined
- Personal and intimate
- Premium and polished

## Deliverables
1. Updated color palette in CSS variables
2. New typography hierarchy with serif/sans-serif combination
3. Refined component designs (cards, buttons, backgrounds)
4. Elegant petal and particle animations
5. Premium gradient effects and lighting
6. Updated imagery/illustrations (elegant birthday cake)
7. Fully responsive design
8. Smooth transitions and animations throughout
